# Terraform Docker ECR workflow

## Steps to run
  - cd 24-terraform-docker-ecr-workflow
  - terraform init
  - terraform validate
  - terraform plan
  - terraform apply --auto-approve
  - terraform destroy --auto-approve
  - rm -r .terraform*
  - rm -r terraform.tfstate*
