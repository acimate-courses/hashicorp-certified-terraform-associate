# Terraform Cross Acc Lambda

## Steps to run
  - cd 25-terraform-cross-acc-lambda
  - terraform init
  - terraform validate
  - terraform plan
  - terraform apply --auto-approve
  - terraform destroy --auto-approve
  - rm -r .terraform*
  - rm -r terraform.tfstate*
